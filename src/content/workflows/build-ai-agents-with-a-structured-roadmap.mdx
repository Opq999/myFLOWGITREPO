## What you'll get

You will learn the fundamental concepts of AI agents, including their basic loop and when they are most effective. This workflow guides you through building a minimal agent capable of using tools, setting the foundation for more complex agent development.

This approach works by breaking down the complex topic of AI agents into manageable stages, starting with core principles and progressing to practical implementation.

## Tools you need

*   **Claude Code** (paid): A coding assistant that can help with generating and understanding code for agent development.
*   **OpenAI** (paid): Provides access to powerful language models and APIs essential for agent functionality.
*   **Gemini API** (freemium): Google's API for accessing their large language models, useful for agent development.
*   **LlamaIndex** (free): A data framework for LLM applications, helpful for building agents that interact with data.
*   **LangChain** (free): A framework for developing applications powered by language models, widely used for building agents.
*   **Claude Tool Use** (paid): Documentation and examples for enabling Claude models to use external tools, via the paid Anthropic API.

## Steps

1.  **Understand the Core Concepts of AI Agents**: Read the provided documentation to grasp the differences between chatbots, workflows, and agents. Understand the agent's observe-think-act loop and when an agent is not the right solution. The author recommends reading Anthropic's "Building effective agents" and OpenAI's "A practical guide to building agents".
    You should gain a clear understanding of what an agent is and its limitations.

2.  **Build a Minimal Agent Loop**: Familiarize yourself with using an LLM API for structured JSON output, defining tool functions (like search or calculator), parsing tool calls, executing tools, and feeding results back to the model. Implement basic error handling and step limits.
    The author recommends referring to documentation for OpenAI Function Calling, Gemini API Function Calling, and Claude Tool Use.
    You should aim to produce a small agent (50-150 lines of code) that can select and use tools to return a final answer.

3.  **Learn Tool Use, RAG, and Memory**: Explore Retrieval Augmented Generation (RAG) concepts, including chunking, embedding, retrieving, and answering with citations. Learn to integrate various tools like search engines, databases, and file systems. Understand different types of memory (short-term, session, long-term) and how to handle tool failures and hallucinations.
    The author suggests exploring LlamaIndex Agents, LangChain Docs, Gemini API Code Execution, and Model Context Protocol.
    You should be able to build a basic research assistant that can search, summarize information, and provide citations for its findings.

4.  **Study a Modern Agent Harness**: Select and deeply study one modern agent system. Focus on how it manages tools, context, permissions, state, logging, sub-tasks, and feedback, rather than just the API calls.
    The author lists several systems to consider, such as Claude Code, OpenClaw, Hermes Agent, and others, each with specific learning goals.
    You should gain a deeper understanding of the engineering principles behind robust AI agents.

## Notes & variations

*   **Cost**: LlamaIndex and LangChain are free, and the Gemini API has a free tier you can learn on. Claude Code, the OpenAI API, and the Anthropic API are paid. Lean on the free options to minimize costs.
*   **Common Pitfall**: A common mistake is getting lost in the vast number of frameworks and tools. It's crucial to follow the structured learning path provided and focus on understanding the core concepts before diving deep into every available option.
*   **Tip for Better Results**: Start by building the "minimal agent loop" as described in Stage 1. Successfully implementing this core functionality will provide a strong foundation and boost confidence for tackling more advanced stages.
