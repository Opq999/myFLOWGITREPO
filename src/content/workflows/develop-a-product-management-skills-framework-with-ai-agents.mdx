## What you'll get

A ready-made, professional product-management toolkit loaded into Claude, so the AI can help you do real product work: frame a problem properly, spot opportunities, and check whether an idea is worth building ("validate" it). The toolkit (an open-source pack of "skills") is built on established PM methods, so you get a structured approach even with no product-management background. No coding needed for the main path.

## Tools you need

- **Claude** (freemium): the AI you load the skills into and chat with. Free tier works, with usage limits.
- **Product Manager Skills** (free): the open-source framework by deanpeters on GitHub. You download it and add it to Claude.

## Steps

1. **Download the starter pack**: on the GitHub repository, find the "Fastest Path" download link and get `pm-skills-starter-pack.zip`. You should have a ZIP file on your device.

2. **Unzip it**: extract the pack. Inside you will find several individual skill ZIPs grouped by area (for example discovery, strategy, delivery, and AI-PM).

3. **Add the skills to Claude**: open Claude (web or desktop) and upload the individual skill ZIPs. (Claude "skills" are add-on capabilities you give it.) Important: upload the individual skill files, not the whole starter-pack ZIP. The download includes an install guide (like `INSTALL-CLAUDE-DESKTOP.md`) if you need the exact clicks.

4. **Start a product task**: prompt Claude to apply the framework to a real problem of yours:

   ```
   Use the Product Manager Skills to help me frame this product problem: [describe your
   product, who it is for, and the challenge you are facing].
   ```

   You should get a structured, method-based response rather than generic advice.

5. **Add more packs as you go**: upload the other skill ZIPs (discovery, strategy, delivery) when you need those capabilities.

6. **Optional, programmatic use (advanced)**: the repo also includes a package for use with OpenAI's Codex coding agent, for people who want to run the framework in code rather than in Claude's chat. This needs technical setup.

## Notes & variations

- **Free-tier viability**: Claude's free tier works but has usage limits. The framework itself is free.
- **Common pitfall**: uploading the whole starter-pack ZIP instead of the individual skill ZIPs inside it. Claude needs the individual ones to use the skills.
- **Tip for better results**: give Claude real context in your first prompt, the product, the audience, and the specific challenge, so its advice is tailored and actionable.
