## What you'll get

This workflow provides a structured, visual, and example-driven guide to mastering Claude Code. You will learn to combine features like slash commands, hooks, memory, and subagents into powerful AI workflows with ready-to-use templates. This approach is ideal for moving beyond basic prompts to build complex, automated tasks.

## Tools you need

*   **Claude Code** (freemium): A powerful AI model for building and orchestrating agents and workflows.
*   **Claude** (freemium): The conversational interface for interacting with Claude Code.

## Steps

1.  **Access the Claude Code How-To Repository**: Navigate to the GitHub repository for the Claude Code How-To guide. You should see a list of folders and files, including documentation and examples.

    *What you should see:* A file listing for the `luongnv89/claude-howto` repository.

2.  **Review the Learning Roadmap**: Locate and open the `LEARNING-ROADMAP.md` file. This document outlines a structured path for learning Claude Code concepts, from basic to advanced.

    *What you should see:* A guide detailing the recommended order to learn different Claude Code features and concepts.

3.  **Explore Basic Concepts**: Navigate to the `01-slash-commands` folder and open the relevant documentation or example files. This section introduces fundamental Claude Code features.

    *What you should see:* Examples and explanations of how to use basic Claude Code commands.

4.  **Understand Memory and Skills**: Proceed to the `02-memory` and `03-skills` folders. Study the provided examples and documentation to learn how Claude Code can retain context and utilize specialized skills.

    *What you should see:* Demonstrations of how to implement memory for context and integrate skills into your workflows.

5.  **Learn About Subagents and MCP**: Explore the `04-subagents` and `05-mcp` folders. These sections cover how to break down complex tasks by using multiple agents and the Message Communication Protocol (MCP).

    *What you should see:* Examples and explanations for creating and coordinating multiple AI agents for more sophisticated tasks.

6.  **Examine Advanced Features and Plugins**: Investigate the `07-plugins` and `09-advanced-features` folders. This step involves understanding how to extend Claude Code's capabilities with external tools and advanced techniques.

    *What you should see:* Documentation and examples for integrating plugins and utilizing advanced Claude Code functionalities.

7.  **Practice with Copy-Paste Templates**: Throughout your learning, utilize the provided copy-paste templates found within the various example folders. These templates offer immediate value and can be adapted for your projects.

    *What you should see:* Ready-to-use code snippets and configurations that you can directly implement.

8.  **Interact with Claude**: Open your Claude interface and begin experimenting with the concepts and templates you've learned. You can start by trying simple prompts related to the features you've studied.

    *What you should see:* Claude's responses to your prompts, demonstrating the application of Claude Code features.
