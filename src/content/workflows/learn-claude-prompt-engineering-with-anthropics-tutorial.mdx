## What you'll get

This workflow guides you through Anthropic's interactive prompt engineering tutorial. You will learn the fundamental structure of effective prompts for Claude models, recognize and fix common prompt failures, and gain the ability to construct strong prompts for various use cases.

## Tools you need

* Claude 3 Haiku (freemium)
* Claude for Sheets (freemium)

## Steps

1. Navigate to the Anthropic Prompt Engineering Interactive Tutorial. The author recommends starting with the 'Claude for Sheets' version for a more user-friendly experience, but the core content is also available directly.
2. Begin with Chapter 1: '01_Basic Prompt Structure'.
3. Work through each lesson and its accompanying exercises. Experiment with the 'Example Playground' area to test how prompt variations affect Claude's responses.
4. Utilize the provided answer key to check your work on the exercises.
5. Proceed through the remaining chapters in order, applying the learned techniques to build and refine prompts.

## Original source

This workflow is based on Anthropic's Prompt Engineering Interactive Tutorial, shared by cjbarber on Hacker News. The tutorial is designed to provide a comprehensive, step-by-step understanding of how to engineer optimal prompts for Claude models, with hands-on exercises.

## Notes & variations

The tutorial explicitly mentions using Claude 3 Haiku, the smallest and fastest model. For more advanced capabilities, Claude 3 Sonnet and Claude 3 Opus are available, though they may have different pricing structures or be accessed through different interfaces. The 'Claude for Sheets' version is recommended for ease of use, leveraging Google Sheets' free tier.
