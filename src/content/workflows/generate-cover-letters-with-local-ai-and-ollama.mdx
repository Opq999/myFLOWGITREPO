## What you'll get
You will generate a personalized cover letter ready to copy and paste, based on your resume and a specific job description. This approach works by using a local AI model to process your documents, ensuring privacy and avoiding API costs, making it ideal for frequent job applications.

## Tools you need
*   **Ollama** (free) - To run large language models locally on your computer.
*   **Cover Letter Maker (Web App)** (free) - A web application that connects to local LLMs to generate cover letters.
*   **LM Studio** (free) - An alternative tool to Ollama for running LLMs locally.

## Steps
1.  **Set up a local LLM** - Download and install Ollama or LM Studio. Follow their respective instructions to download a model like Llama 3.2. Ensure your local LLM server is running.
2.  **Access the Cover Letter Maker web app** - Open the Cover Letter Maker web application. The author mentions it can be run locally or self-hosted, but for ease of use, you can likely access a hosted version or follow their GitHub instructions to run it on your machine.
3.  **Upload your resume** - On the web app, find the option to upload your resume. Select your resume file (it should be a PDF). The app will parse and cache your resume data in your browser.
    You should see a confirmation that your resume has been processed.
4.  **Paste the job description** - Locate the text area for the job description. Copy the full text of the job description you are applying for and paste it into this area.
    The text area should now contain the job description.
5.  **Generate the cover letter** - The application automatically connects to your running local LLM (like Ollama). Click the button to generate the cover letter. The author states this takes approximately 5 seconds.
    You should see a complete, personalized cover letter appear in an output box, ready to be copied.
6.  **Copy and use the cover letter** - Click the one-click copy button provided by the web app to copy the generated cover letter to your clipboard. You can then paste it into your job application or email.

## Original source
This workflow is based on a web application created by stanyy, shared on Hacker News. The tool aims to provide a private and efficient way to generate cover letters using locally run AI models, addressing the author's frustration with placeholder-filled templates and API costs.

## Notes & variations
*   **Free-tier alternatives**: If you prefer not to set up local LLMs, you could use free tiers of online AI writing tools, but be mindful of their privacy policies and usage limits. However, this workflow's core benefit is local, private generation.
*   **Common pitfall**: Ensure your local LLM (Ollama, LM Studio) is running and accessible before attempting to generate the cover letter. If the web app cannot connect, it won't be able to generate text.
*   **Tip for better results**: While the tool parses your resume, reviewing the generated cover letter for tone and specific keywords from the job description before sending can further improve its effectiveness.
