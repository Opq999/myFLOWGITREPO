## What you'll get
You will set up an AI-powered system that monitors public opinion and trends across various online platforms. This tool aggregates news, filters it based on your keywords, and provides AI-driven analysis and smart alerts, helping you stay informed without information overload.

## Tools you need
*   **TrendRadar** (free) - The core AI tool for monitoring and analyzing public opinion and trends.
*   **Docker** (freemium) - A platform to easily deploy and run applications like TrendRadar.
*   **Telegram, Email, or Slack** (free/freemium) - Notification channels to receive smart alerts and analysis directly.

## Steps
1.  **Clone the TrendRadar repository** - Navigate to the GitHub page for TrendRadar and click the 'Code' button to copy the repository URL. Then, open your terminal or command prompt and use the `git clone` command followed by the URL.
    ```bash
    git clone https://github.com/sansan0/TrendRadar.git
```
    You should see the TrendRadar project files downloaded to your local machine.

2.  **Install Docker** - If you don't have Docker installed, download and install it from the official Docker website. Follow the on-screen instructions for your operating system.
    You will have Docker Desktop or Docker Engine running on your system.

3.  **Navigate to the TrendRadar directory** - In your terminal, change your directory to the newly cloned TrendRadar folder.
    ```bash
    cd TrendRadar
```
    Your terminal prompt should now show you are inside the TrendRadar directory.

4.  **Build the Docker image** - Use Docker to build the application image. The exact command might vary slightly based on the project's setup, but a common command is:
    ```bash
    docker build -t trendradar .
```
    You should see Docker compiling the image, which may take a few minutes. Look for a message indicating the build was successful.

5.  **Start the TrendRadar service** - Run the Docker container to start the TrendRadar application. The `docker run` command will start the service.
    ```bash
    docker run -d --name trendradar_container trendradar
```
    You should see a container ID printed, and the application will start running in the background.

6.  **Configure your alert channels** - Access the TrendRadar configuration files (details are in the project's README) to set up your preferred notification channels like Telegram, Email, or Slack. This typically involves adding API keys or webhook URLs for your chosen service.
    The configuration files will be updated with your specific channel details.

7.  **Set up keywords for monitoring** - Within the TrendRadar configuration, specify the keywords or topics you want to monitor. This tells the AI what public opinion and trends to focus on.
    Your chosen keywords will be saved in the configuration, and the monitoring will begin.

8.  **Receive AI-generated alerts and analysis** - Once configured, TrendRadar will start aggregating news, filtering it, and sending summarized analysis and alerts to your chosen channels.
    You will begin receiving notifications on your Telegram, Email, or Slack with insights on the trends you are monitoring.

## Notes & variations
*   **Free Tier Viability**: TrendRadar itself is free and open-source. The primary cost consideration would be any associated API usage fees for data sources if not using the default ones, and the notification services (Telegram, Email, Slack) generally have generous free tiers suitable for individual or small business use.
*   **Common Pitfall**: Ensure your Docker installation is correct and that you have sufficient system resources (RAM, disk space) to run the Docker container. Incorrect Docker commands or insufficient resources can prevent the application from starting.
*   **Tip for Better Results**: Experiment with different keyword combinations and monitor frequencies. The AI's analysis quality can be significantly improved by providing precise keywords and adjusting how often the system scans for new information.
