## What you'll get

You will learn how to convert a YOLOv5 object detection model from its native PyTorch format into ONNX, CoreML, and TensorFlow Lite (TFLite) formats. This is crucial for deploying your trained models on various platforms and devices, from web applications to mobile apps.

## Tools you need

*   **Ultralytics YOLOv5** (freemium) - The object detection model framework. You'll use its `export.py` script.
*   **PyTorch** (free) - The deep learning framework YOLOv5 is built upon.
*   **ONNX** (free) - An open format to represent machine learning models, enabling interoperability.
*   **Core ML** (free) - Apple's framework for integrating machine learning models into applications.
*   **TensorFlow Lite (TFLite)** (free) - A framework for deploying TensorFlow models on mobile, embedded, and IoT devices.

## Steps

1.  **Clone the YOLOv5 repository**: If you haven't already, clone the official YOLOv5 repository from GitHub to your local machine. This repository contains the necessary scripts for model conversion.
    *   Open your terminal or command prompt.
    *   Run the following command:

    ```bash
    git clone https://github.com/ultralytics/yolov5
    ```

    *   You should see output indicating the repository is being cloned, and a new `yolov5` directory will be created.

2.  **Navigate to the YOLOv5 directory**: Change your current directory to the newly cloned `yolov5` folder.
    *   In your terminal, run:

    ```bash
    cd yolov5
    ```

    *   Your terminal prompt should now show that you are inside the `yolov5` directory.

3.  **Install dependencies**: Install all the required Python packages for YOLOv5 to function correctly. This includes PyTorch and other libraries.
    *   Ensure you have Python 3.8.0 or newer and PyTorch 1.8 or newer installed.
    *   Run the following command in your terminal:

    ```bash
    pip install -r requirements.txt
    ```

    *   You will see a list of packages being downloaded and installed. This might take a few minutes.

4.  **Export the model to ONNX**: Use the `export.py` script to convert a pre-trained YOLOv5 model (e.g., `yolov5s.pt`) to the ONNX format. The script will automatically download the model if it's not found locally.
    *   Run the following command in your terminal (from within the `yolov5` directory):

    ```bash
    python export.py --weights yolov5s.pt --include onnx
    ```

    *   The script will output progress messages, including downloading the weights if necessary, and then indicate that the ONNX model has been exported. You should find a `yolov5s.onnx` file in the `yolov5/weights` directory.

5.  **Export the model to Core ML**: Similarly, export the model to the Core ML format.
    *   Run the following command:

    ```bash
    python export.py --weights yolov5s.pt --include coreml
    ```

    *   You should see output confirming the Core ML export, and a `yolov5s.mlmodel` file will be available in the `yolov5/weights` directory.

6.  **Export the model to TFLite**: Export the model to the TensorFlow Lite format.
    *   Run the following command:

    ```bash
    python export.py --weights yolov5s.pt --include tflite
    ```

    *   The script will confirm the TFLite export, and you will find a `yolov5s.tflite` file in the `yolov5/weights` directory.

## Notes & variations

*   **Free Tier Viability**: All the core tools mentioned (PyTorch, ONNX, Core ML, TFLite) are free and open-source. The YOLOv5 framework itself is also free to use for most purposes. The primary cost is your time and potentially data for downloading.
*   **Common Pitfall**: Ensure you have the correct versions of PyTorch and Python installed as specified in the `requirements.txt` file. Incompatibilities can lead to errors during the export process.
*   **Tip for Better Results**: You can export to multiple formats simultaneously by listing them with the `--include` argument, for example: `python export.py --weights yolov5s.pt --include onnx coreml tflite`. This saves you from running the export command multiple times for different formats.
