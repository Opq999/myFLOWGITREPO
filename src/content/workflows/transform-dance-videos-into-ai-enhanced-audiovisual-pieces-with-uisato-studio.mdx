## What you'll get
You will transform an ordinary iPhone recording of a dance performance into a multi-angle, AI-enhanced audiovisual piece. This workflow allows you to apply a unique aesthetic, such as a robot or bio-tech look, without expensive equipment or a large production budget.

## Tools you need
*   iPhone (paid)
*   Uisato Studio (freemium)
*   Gemini (freemium)
*   Editing software (freemium)

## Steps
1.  Record your original dance performance video using an iPhone.
2.  Split the original dance/performance video into clips. Use 3-10 second clips if you want to use a new camera angle from a reference image, or up to 30 seconds if you want to preserve the original camera angle from the video source.
3.  Open Uisato Studio and select the Kling Motion Control mode.
4.  For each clip, provide the original performance video as the reference video input.
5.  For each section, provide a target image with the desired aesthetic (e.g., robot/bio-tech) as the reference image.
    *   You can use Uisato Studio's "capture frame" function to intervene one of the input video's frames using Gemini.
    *   Alternatively, you can bring your own pre-intervened reference images. This is where you can introduce different points-of-view for the intervened scene.
6.  Craft a brief, balanced prompt describing what you want beyond the motion transfer. The author doesn't share their exact prompt; a starting point:

    
an avant-garde humanoid android performer dancing (...)
    

    Another example prompt:

    
you might introduce subtle robotic precision while still following the original dance (...)
    

7.  Select the Kling-3 model. The author notes that while the "std" Kling-3 model performs really well, they went with "pr..." (the rest of the model name is not provided in the source).
8.  Generate the interventions using Uisato Studio.
9.  Use editing software to assemble the generated clips into a complete multi-angle audiovisual piece.

## Notes & variations
Uisato Studio likely offers a free tier for basic use, but extensive video generation may require a paid plan. Gemini has a robust free tier. For editing software, free alternatives like DaVinci Resolve (desktop) or CapCut (mobile/desktop) can be used instead of paid options. Video processing, especially with AI, can consume significant data. Consider using Wi-Fi for generation and uploads/downloads.
