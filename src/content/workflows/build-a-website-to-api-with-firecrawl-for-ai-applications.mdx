## Notes & variations
*   **Free-tier alternatives:** For basic web scraping without an API, you can use Python libraries like `BeautifulSoup` or `Scrapy`. These require more coding but are completely free. Other freemium services like Apify or ScrapingBee also offer similar functionalities.
*   **Common mistake:** Forgetting to check a website's `robots.txt` file or terms of service before scraping. Always respect website policies and avoid excessive requests that could lead to your IP being blocked.
*   **Tip for better results:** When scraping, start with well-structured, text-heavy pages to understand FireCrawl's output. For more complex sites, FireCrawl often provides options for different output formats (e.g., markdown, raw HTML) which you can explore to get the best data for your specific AI task.
