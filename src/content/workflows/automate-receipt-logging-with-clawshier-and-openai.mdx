## What you'll get
You will create an automated system that takes a picture of a receipt and logs its details, such as establishment name, category, and total amount, into an expense spreadsheet. This approach is useful for small businesses and freelancers looking to streamline expense tracking and reduce manual data entry.

## Tools you need
*   **Clawshier** (free): An open-source tool designed to automate receipt processing.
*   **OpenAI API** (paid): A service that provides AI models for tasks like image recognition and text extraction.
*   **GitHub** (free): A platform for hosting and collaborating on code, where Clawshier is available.

## Steps
1.  **Install Clawshier**: Follow the installation instructions provided in the Clawshier GitHub repository. This typically involves cloning the repository and setting up any necessary dependencies.
    You should see the Clawshier project files successfully downloaded to your local machine.
2.  **Configure OpenAI API Key**: Obtain an API key from OpenAI. You will need to configure Clawshier to use this key. The exact method for configuration is not detailed in the source, but it usually involves setting an environment variable or editing a configuration file.
    Clawshier should now be authenticated to use OpenAI's services.
3.  **Prepare Your Receipt Image**: Take a clear picture of your receipt. Ensure good lighting and that all text is legible. For best results, try to capture the entire receipt without significant glare or shadows.
    You will have a digital image file of your receipt ready for processing.
4.  **Process the Receipt**: Use the Clawshier tool to process the receipt image. The author does not specify the exact command, but it will likely involve running a script and providing the path to your receipt image file.
    The tool will attempt to extract information from the receipt. You should see output indicating the processing status and any extracted data.
5.  **Review Extracted Data**: Examine the data extracted by Clawshier. The tool aims to identify the establishment name, expense category, totals, and taxes. Be aware that date extraction can be inconsistent due to different date formats (DD-MM vs MM-DD).
    You will have a list of extracted details, which you can then manually verify or use as a basis for the next step.
6.  **Update Expense Spreadsheet**: Clawshier automatically creates or updates several spreadsheet files, including a Summary, Invoice Archive, and Monthly expense sheet. The extracted data is logged into these files.
    Your expense tracking spreadsheets will be updated with the new receipt information.

## Notes & variations
*   **Cost**: Clawshier is free and open-source. The OpenAI API is pay-as-you-go (no free tier); light use is inexpensive but costs increase with higher volumes. Consider local AI models (like Ollama with Llama 3.2 Vision) for free processing and data privacy, though they require more setup and potentially more RAM.
*   **Common Pitfall**: Inconsistent date formats on receipts can lead to errors in date extraction. The author suggests manually specifying the date as a workaround if you encounter this issue.
*   **Better Results**: For improved image recognition accuracy, especially with thermal receipts or cryptic text, ensure your receipt photos are as clear and well-lit as possible. Experimenting with different image preprocessing techniques or alternative OCR engines like PaddleOCR or Tesseract could also enhance results.
