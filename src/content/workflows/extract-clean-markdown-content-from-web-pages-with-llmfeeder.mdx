## What you'll get
You will be able to extract the main content from any web page as clean markdown, ready to be pasted into AI coding assistants. This approach works by stripping away website clutter like ads and navigation, ensuring your AI receives only the relevant text for better context and responses.

## Tools you need
*   **LLMFeeder** (free) — A browser extension that extracts and converts web page content to markdown.
*   **Mozilla Readability.js** (free, included in LLMFeeder) — The underlying technology that identifies and extracts the main content of a web page.
*   **Turndown.js** (free, included in LLMFeeder) — A library that converts HTML to markdown.
*   **AI Coding Assistant** (freemium, e.g., ChatGPT, Cursor) — A tool where you will paste the extracted markdown content.

## Steps
1.  **Install LLMFeeder** — Go to your browser's extension store (Chrome Web Store or Firefox Add-ons) and search for "LLMFeeder". Click "Add to [Browser]" and confirm the installation. You should see the LLMFeeder icon appear in your browser's toolbar.
2.  **Navigate to a web page** — Open any web page containing documentation or article content you want to extract. For example, visit a technical blog post or a software documentation page.
3.  **Activate LLMFeeder** — On the web page, press the keyboard shortcut `Alt+Shift+M` (on Windows/Linux) or `Option+Shift+M` (on macOS). You should hear a subtle sound or see a brief visual confirmation indicating that the content has been copied to your clipboard.
4.  **Paste into your AI assistant** — Switch to your AI coding assistant (like ChatGPT or Cursor). Create a new chat or prompt and paste the content directly using `Ctrl+V` (Windows/Linux) or `Cmd+V` (macOS). The pasted text should be well-formatted markdown, free of ads and navigation elements.
5.  **Use the context** — You can now ask your AI assistant questions about the pasted content, request summaries, or ask it to perform tasks based on the provided documentation.

## Notes & variations
*   **Free-tier alternative**: LLMFeeder itself is free and open-source. For AI assistants, free tiers of tools like ChatGPT or Gemini can be used, though they may have usage limits.
*   **Common pitfall**: Ensure you are on the actual content page and not a link aggregation page or a page with complex dynamic content that might confuse the extractor. Sometimes, refreshing the page before using the shortcut can help.
*   **Tip for better results**: If the extracted markdown isn't perfect, you can configure LLMFeeder's settings (accessible via its extension icon) for content scope and formatting options. Experimenting with these settings can yield cleaner results for specific websites.
