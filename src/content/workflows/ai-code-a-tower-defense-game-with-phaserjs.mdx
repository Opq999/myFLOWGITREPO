## What you'll get
You will learn how to leverage AI coding agents to assist in developing a browser-based tower defense game using the Phaser.js engine. This workflow provides insights into the entire development process, from initial coding to asset integration and submission.

## Tools you need
* Phaser.js (freemium)
* AI coding agents (freemium)
* itch.io (freemium)
* freesound.org (free)

## Steps
1. Learn the basics of Phaser.js, a Javascript-based game engine.
2. Utilize AI coding agents to assist in programming the game.
3. Source art assets from platforms like itch.io, ensuring free usage rights.
4. Find sound effects from freesound.org.
5. Integrate art and sound assets into the Phaser.js game.
6. Document the entire development process, including code and prompts used.
7. Submit the completed game to a platform like itch.io.

## Original source
This workflow is inspired by a software developer with over 20 years of experience who documented their journey of creating a tower defense game called "Tower of Time" using AI coding agents and the Phaser.js engine. The process, including code and prompts, was shared on GitHub, with art assets sourced from itch.io and sounds from freesound.org.

## Notes & variations
* For AI coding assistance, consider free tiers of models like ChatGPT or Gemini. Be mindful of usage limits.
* Explore other free JavaScript game engines if Phaser.js does not meet your needs.
* For art assets, look for open-source or Creative Commons licensed resources.
* The author streamed parts of their development process, which could be a valuable resource for visual learners.
